# Vault Drift — 2D browser game

A small self-contained HTML5 canvas game. No install, no build step.

## Play

```bash
# from this folder
open index.html
# or
python3 -m http.server 8765
# then visit http://127.0.0.1:8765
```

Or double-click `index.html` in your file manager.

## Controls

| Action | Keys |
|--------|------|
| Move | **WASD** or **Arrow keys** |
| Shoot | **Space** or **Left click** |
| Dash | **Shift** |
| Pause | **P** / **Esc** |

Touch: drag to move, hold to shoot.

## Goal

Survive waves of corrupted packets, build combos for score multipliers, grab data cores (heal / bonus score). Best score is saved in the browser.
